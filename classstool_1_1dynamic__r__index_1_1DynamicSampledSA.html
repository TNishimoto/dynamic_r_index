Detailed Description

O(n log n / k)-bit dynamic data structure for accessing SA[0..n-1] and ISA[0..n-1] sampled by the sampling interval k.

This data structure consists of the following three components:

  • Dynamic bit sequence B[0..n-1]: B[i] = 1 if and only if SA[i] is a sampled value.
  • Dynamic bit sequence F[0..n-1]: F[i] = 1 if and only if i is a sampled SA value.
  • Dynamic permutation Π[0..n/k]: Let j be the (i+1)-th sampled position in SA. Then Π[i] stores the number of 1 in F[0..SA[j]-1]. Here, SA[j] = select1_{F}(Π[i]) holds.
