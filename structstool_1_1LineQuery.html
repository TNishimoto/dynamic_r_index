Detailed Description

Structure representing a single query from a command file.

This structure stores the query type and parameters parsed from a TSV command file line.

Parse a line from a command file into a LineQuery.

Parameters
lineThe line to parse (TSV format)
alternative_tab_keyAlternative tab character for parsing
alternative_line_break_keyAlternative line break character for parsing
Returns
A LineQuery parsed from the line, or NONE query if parsing fails

Supported formats:

  • INSERT\t<position>\t<string>
  • DELETE\t<position>\t<length>
  • COUNT\t<pattern>
  • LOCATE\t<pattern>
  • LOCATE_SUM\t<pattern>
  • PRINT
