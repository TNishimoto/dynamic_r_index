Detailed Description

A dynamic data structure for stroing two arrays C[0..σ-1] and D[0..σ'-1] related to an alphabet Σ = {c_{1}, c_{2}, ..., c_{σ-1}} of a string T[0..n-1].

The details of this data structure are as follows:

  • Each C[i] stores the number of occurrences of c_{1}, c_{2}, ..., c_{i} in T[0..n-1].
  • Each D[i] stores the i-th character c'_{i} in Σ'. Here, Σ = {c'_{1}, c'_{2}, ..., c'_{σ'-1}} is a set of characters that appear in the string T[0..n-1].
