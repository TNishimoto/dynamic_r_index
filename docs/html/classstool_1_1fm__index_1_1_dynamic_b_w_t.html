Detailed Description

A dynamic data structure storing BWT. This implementation requires $O(n log σ)$ bits of space for the input string of length $n$ and alphabet size $σ$.

void stool::fm_index::DynamicBWT::initialize ( const std::vector< uint8_t > & _alphabet)
inline

Initializes the DynamicBWT with the given characters and end marker.

This function sets up the BWT (Burrows-Wheeler Transform) structure with the provided characters. It ensures the end marker is included, initializes the dynamic rank-select structure, and updates the character distribution and C array.
